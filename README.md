# Miten

Improve your meetings by getting feedback

[Work in progress]

### How to run

#### development

(This will change, but for now:)

First install the packages and init the environment variables:

```
npm i
cd ./backend
npm i
cp ./.env.sample ./.env
cd ../frontend
npm i
```

Run backend:

```
cd ./backend
docker-compose up -d
npm start
```

Run frontend:

```
cd ./frontend
npm start
```
